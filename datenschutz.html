2. Datenerfassung auf dieser Website

Wer ist verantwortlich für die Datenerfassung auf dieser Website?
Die Datenverarbeitung auf dieser Website erfolgt durch den Websitebetreiber. Dessen Kontaktdaten können Sie dem Impressum dieser Website entnehmen.

Wie erfassen wir Ihre Daten?
Auf dieser Website werden keine personenbezogenen Daten automatisch erfasst. Personenbezogene Daten werden nur dann erhoben, wenn Sie uns diese freiwillig mitteilen, beispielsweise indem Sie uns eine E-Mail senden oder telefonisch kontaktieren. In diesem Fall verwenden wir die übermittelten Daten ausschließlich zur Bearbeitung Ihrer Anfrage.
